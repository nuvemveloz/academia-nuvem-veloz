1
Hospedar os arquivos no GitHub Pages
Gratuito, permanente, sem limite de acessos — gera URLs públicas para cada material
github.com
1.1

🌐 Crie uma conta gratuita no GitHub

Acesse github.com e clique em "Sign up". Crie com seu e-mail normal — é 100% gratuito para este uso.

💡

Se já tiver conta no GitHub, pule para o próximo passo.

1.2

📂 Crie um novo repositório

Após o login, clique no botão verde "New" (ou no "+" no canto superior direito → "New repository").

Preencha assim:
Repository name:academia-nuvem-veloz
Visibility:✅ Public (obrigatório para GitHub Pages gratuito)
Add README:✅ marque esta opção

Clique em "Create repository".

1.3

⬆️ Faça upload dos arquivos HTML

Dentro do repositório criado, clique em "uploading an existing file" (ou "Add file" → "Upload files").

Arraste os 6 arquivos .html do curso para a área de upload:

Arquivos para fazer upload:
📄 academia-jovens-empreendedores.html
📄 jogo-narrativo-academia.html
📄 apostila-jovens-empreendedores.html
📄 certificado-academia.html
📄 planilha-acompanhamento.html
📄 ikigai-jovens.html

Clique em "Commit changes" para salvar.

1.4

⚙️ Ative o GitHub Pages

No repositório, clique em "Settings" (engrenagem) → no menu lateral, clique em "Pages".

Configure assim:
Source:Deploy from a branch
Branch:main / root

Clique em "Save". Aguarde 1–2 minutos. Sua URL estará disponível assim:

https://seu-usuario.github.io/academia-nuvem-veloz/academia-jovens-empreendedores.html
✅

Teste abrindo a URL no navegador. Se o arquivo aparecer, está funcionando! A partir de agora, qualquer arquivo que subir aparecerá nessa URL.

1.5

📋 Anote todas as URLs geradas

Cada arquivo vira uma URL. Anote todas para usar no GrooveMember:

Hub Principal: .../academia-jovens-empreendedores.html
Jogo Narrativo: .../jogo-narrativo-academia.html
Apostila: .../apostila-jovens-empreendedores.html
Certificado: .../certificado-academia.html
Planilha: .../planilha-acompanhamento.html
Ikigai: .../ikigai-jovens.html
💡

O botão de imprimir do certificado e todos os outros recursos funcionarão perfeitamente nestas URLs públicas — o problema que existia localmente não ocorre quando hospedado.

2
Criar a página de vendas e o produto no Groove
GroovePages (landing page) + GrooveSell (checkout e pagamento)
groove.cm
2.1

🛒 Criar o produto no GrooveSell

No painel do Groove.cm, acesse GrooveSell → Products → Add New Product.

Preencha as informações do produto:
Product Name:Academia dos Jovens Empreendedores
Type:Digital Product / Course
Price:R$ [seu preço]
Currency:BRL (Real Brasileiro)

Na aba "Thank You Page", coloque a URL da sua área de membros (configurada na Etapa 3).

2.2

💰 Sugestão de preço

O curso tem 32 aulas, apostila, jogo, certificado e acompanhamento. Considere:

ModalidadePreço sugeridoPúblico
Turma presencial — por aluno/mêsR$ 350–500/mêsIndividual ou dupla
⭐ Turma presencial — pacote completo (4 meses)R$ 997–1.497Pagamento único (melhor conversão)
Acesso online aos materiaisR$ 197–397Autoestudo com suporte
💡

Para começar, cobre pela turma presencial. O produto digital pode ser criado depois, quando tiver depoimentos e provas do resultado.

2.3

📄 Criar a página de vendas no GroovePages

Acesse GroovePages → + New Website → escolha um template (ou comece em branco). Recomendo o template de "Course" ou "Online Program".

A página de vendas precisa ter:

Estrutura da página de vendas (seções obrigatórias):
1. 🎯 Hero: título impactante + para quem é + botão de compra
2. 🤔 O problema: por que empreendedorismo para jovens?
3. ✨ A solução: o que é a Academia + foto/vídeo
4. 📚 O que está incluso: lista dos materiais
5. 🗺️ As 6 fases: visão geral do conteúdo
6. 💬 Depoimentos (após ter alunos)
7. 💰 Preço + botão de compra
8. ❓ FAQ
💡

Posso escrever o texto completo da página de vendas para você — só me pedir! É só um novo chat com "escreva o copy da página de vendas da Academia."

2.4

🔗 Conectar o botão de compra ao GrooveSell

No editor do GroovePages, clique no botão de compra → "Link" → selecione "GrooveSell Checkout" → escolha o produto criado no passo 2.1.

O Groove cuida de todo o checkout automaticamente — boleto, PIX, cartão de crédito.

3
Criar a área de membros no GrooveMember
Onde o aluno acessa os materiais após a compra — login protegido
GrooveMember
3.1

🔐 Criar a membership

No Groove, acesse GrooveMember → + Create Membership.

Configure assim:
Name:Academia dos Jovens Empreendedores
Products:Selecione o produto criado no GrooveSell
Access:Full Access (acesso imediato após compra)
3.2

📁 Criar as seções de conteúdo

Dentro da membership, crie seções para organizar os materiais. Clique em "+ Add Section" para cada seção:

Seções recomendadas:
📌 Começar aqui — Boas-vindas + instruções
🎮 Hub Principal — Link para o hub das 32 aulas
🕹️ Jogo Narrativo — Link para o jogo
📚 Apostila — Link para imprimir a apostila
✨ Teste Ikigai — Link para o teste
🎓 Certificado — Link para gerar o certificado
📊 Planilha — Link para o modelo de acompanhamento
3.3

🔗 Adicionar os links do GitHub Pages

Em cada seção, clique em "+ Add Lesson". No tipo de conteúdo, selecione "External URL" e cole a URL correspondente do GitHub Pages.

💡

O aluno clica no link e o material abre diretamente no navegador. Como os arquivos já estão hospedados no GitHub Pages, funcionam perfeitamente sem precisar de download.

⚠️

Não use "Embed" (iframe) para os arquivos grandes — use sempre "External URL" para abrir numa nova aba. Os arquivos HTML são grandes e embutir pode ficar lento.

3.4

📧 Configurar e-mail automático de boas-vindas (GrooveMail)

No GrooveMail, crie uma automação: Gatilho: "Membro adicionado ao produto X" → Ação: Enviar e-mail.

Conteúdo do e-mail de boas-vindas:
Assunto:🎮 Bem-vindo à Academia, [nome]! Sua missão começa agora
Corpo:Link para a área de membros + instrução de primeiro acesso
4
Conectar seu domínio (opcional, mas profissional)
Usar academia.nuvemveloz.com.br em vez do endereço do GitHub ou Groove
Opcional
4.1

🌐 Você já tem domínio?

Se já tem nuvemveloz.com.br, pode criar um subdomínio como academia.nuvemveloz.com.br e apontar para o Groove.

No painel do seu domínio (registro.br, GoDaddy, etc.), crie um registro CNAME:

Nome/Host:academia
Tipo:CNAME
Valor/Destino:seu-site.groove.page

Depois, no Groove → Settings → Custom Domain, adicione academia.nuvemveloz.com.br.

💡

Se não tem domínio ainda, o endereço do Groove (academia-nuvem-veloz.groove.page) funciona perfeitamente para começar vender. Você conecta o domínio depois.
